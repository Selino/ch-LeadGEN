# Documentation DEMO for the LeadGen Design System

- The purpose of this site is to demonstrate a general ability to integrate the [LeadGen UIKit](https://www.figma.com/file/xGF6TXDWOoigzGthqp9i70/LG-UIKit-v3?node-id=665%3A0) into Storybook.
- This is an ongoing project tied to the [LeadGen React DEMO](https://ch-leadgen-demo.netlify.app)

## About

Storybook is the heart of the LeadGen Design System. It is the famed **single source of truth** where designers, developers, and stake holders all find the documentation needed to craft components.

> **Design System**
> : a collection of reusable components, guided by clear standards, that can be assembled together to build any number of applications
